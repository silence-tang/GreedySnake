一、版本说明
1.Greedy Snake双人版就是在单人版的基础上新增了由另一名玩家控制的贪吃蛇。
2.一条蛇由w s a d控制上下左右移动方向，另一条蛇由↑ ↓ ← →控制移动方向
3.在游戏中，玩家1和玩家2分别操控两条贪吃蛇去觅食，每次吃到一个食物该玩家的得分+1。
4.若玩家1的贪吃蛇在运动过程中蛇头触碰到了玩家2的贪吃蛇蛇身，则玩家1死亡，
玩家2可继续操控他的贪吃蛇直到他也死亡，显示游戏结束界面和最终玩家1,2的得分，反之亦然。
二、和单人版的不同之处
1.show_end函数需要显示双方得分；
2.删除了判定贪吃蛇是否死亡的函数并内嵌于start_game主体中；
3.start_game函数中新增了对于第二条贪吃蛇的描述语句；
4.由wsad控制另一条蛇的上下左右移动方向；
三、技术要点
start_game函数的设计，尤其是while循环体内的语句设计，需要考虑清楚细节问题。
主要包括：if语句条件的设定、死亡标志的设定、死亡判定规则设计、死亡之后的数据销毁及重新绘制画面的操作
